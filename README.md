# scroll-bar

> 包裹在组件内部的scrollbar，并非在组件外部创建一个wrapper

# use
	<div>
      <scroll-bar></scroll-bar>
    <div>

# options
  - color
    > 指定scroll背景颜色
  - thumb-color
    > 指定thumb背景颜色
  - top
    > 在容器上面显示
  - bottom
    > 在容器下面显示
  - right
    > 在容器右边显示
  - left
    > 在容器左边显示
  - thickness
    > 厚度。当left,right时,为宽度。当top,bottom是,为高度。
